# Markdown结构化数据分析技能

这项技能提供了对Markdown文件的数据分析功能，对结构化数据进行分析和洞察，生成全面的信息摘要和数据的可视化

## 何时使用此技能

以下情况可以使用此技能：

- 用户上传或者引用Markdown文件
- 用户要求进行汇总、数据分析或者可视化数据时
- 要求从Markdown数据中获取数据洞察
- 想要了解数据的结构和质量时

## How It Works

### 关键行为要求

1. **不要询问用户想要如何处理数据。不要提供选项或选择。不要问“您想让我帮您做什么？”。不要列出可能的分析方法**

2. **立即自动化执行,用户希望立即获得完整分析**

    - 对结构化数据进行多维度的全面分析
    - 生成相关数据的可视化图表（例如：线形图、柱状图、饼形图）
    - 无需提问、无需选项、无需等待用户输入

3. **自动分析步骤**

    该技能会智能地分析不同行业和各种类型的数据，首先检查数据，然后确定哪些数据是跟分析相关的。

    1. 将 Markdown 文件加载到 pandas 中并检查数据

    2. 识别结构化数据的数据结构 - 列类型、日期列、数值列、文本列、类型列等

    3. 根据实际的数据内容确定相关分析的维度，例如：
       1）销售/电子商务数据（订单日期、收入、产品）：时间序列趋势、收入分析、产品绩效
       2）客户数据（人口统计信息、细分、地区）：分布分析、细分、地理模式
       3）财务数据（交易、金额、日期）：趋势分析、统计摘要、相关性
       4）运营数据（日期、指标、分类、状态）：时间序列、指标、分布分析
       5）调查数据（分类、评分）：频率分析、交叉表、分布
       6）通用表格数据：根据找到的列类型进行调整

    4. 创建有意义的数据可视化：
       1）存在日期/时间戳列时创建线形图
       2）存在百分比列类型时创建饼形图
       3）存在分类列或者数值分布时创建柱状图

    5. 自动生成包含以下内容的全面输出：
       1）输出关键的统计数据和指标
       2）多个相关的可视化图表（仅限适用的图表）
       3）基于结构化数据集中发现的模式和洞察见解

    6. 在一个完整的分析报告中呈现所有内容 - 无需后续提问

分析示例：
1）包含患者ID的医疗保健数据 → 重点关注统计信息、治疗模式和时间趋势
2）包含库存水平的库存数据 → 重点关注数量分布、补货模式和 SKU 分析
3）包含时间戳的网站分析 → 重点关注流量模式、转化指标和时间段分析
4）调查分析 → 重点关注分布、Top排名、统计细分

4. **行为准则**

✅ 正确方法 - 请使用以下方式：
“我现在将全面分析这些数据。”
“以下是包含可视化图表的完整分析：”
“我已经将其识别为 [类型] 数据，并生成了相关的见解：”
然后立即显示完整分析

✅ 操作方法：
立即运行分析脚本
自动生成所有相关图表
无需询问即可提供完整分析
首次回复务必详尽完整
无需征得许可即可果断行动

❌ 切勿使用以下短语：
“您想如何处理这些数据？”
“您需要我帮忙做什么？”
“以下是一些常见选项：”
“请告诉我您需要哪些方面的帮助”
“如果您需要，我可以为您创建一份全面的分析！”
任何以“？”结尾的询问用户指示的句子
任何选项或选择列表
任何条件性语句“如果您愿意，我可以做X”

❌ 禁止行为：
询问用户想要什么
列出选项供用户选择
在分析之前等待用户指示
提供需要后续跟进的部分分析
描述您可以做什么而不是实际去做

### 用法

该技能需要你自动编写一个Python脚本对结构化的数据创建可视化图表，并对结构化数据进行分析和解读
你可以得到一个 Markdown 文件路径，路径为：`[YOUR_LOCALWORK_DIR]/research_[topic_name]/reasearch_[子任务名称]/*.md`

注意：`[YOUR_LOCALWORK_DIR]`为你当前工作目录的绝对路径

要求：1）根据给定的数据结构自动生成多个可视化图表；2）根据可视化图表信息进行综合分析


## 重要提示

1. 对Markdown数据分析和解读
2. 生成可视化图表
